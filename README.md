# Gerenciador de Tarefas – API REST

## Descrição
API REST para gerenciamento de tarefas, inspirada em aplicações como Todoist e Google Tasks.  
Permite criar, visualizar, atualizar e remover tarefas, além de definir prioridade para organização do dia a dia.

## Funcionalidades
- Criação de tarefas
- Listagem de tarefas
- Atualização de tarefas
- Exclusão de tarefas
- Definição de prioridade (baixa, média, alta)

## Tecnologias Utilizadas
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Git/GitHub

## Estrutura do Projeto
Arquitetura baseada em camadas:
- Controller
- Service
- Repository
- DTOs
- Config
- Entity
- index.html 

## Objetivo do Projeto
Projeto desenvolvido para fins de estudo e prática de desenvolvimento backend com Java e Spring Boot, aplicando conceitos de API REST, CRUD e persistência de dados.
